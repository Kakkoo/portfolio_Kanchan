   Filler words counter(React, Redux, JavaScript, NodeJS, MongoDB) deployed via heroku

Deployed via Heroku
Code on github

Having recently joined a Toastmasters club, I realized a need to keep a count of Filler words(Aah, Uhm, Err etc) spoken by various speakers in an online application. This is one of the key functions in every Toastmasters meeting. I created a web-based application, with an easy to use interface to keep a count of Filler words used and report at the end of the meeting.
